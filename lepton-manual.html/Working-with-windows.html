5.7 Working with windows

You can open new and close existing lepton-schematic windows. Schematic data may be exchanged between the windows using system clipboard.

File → New Window (F W) allows you to open a new window of lepton-schematic (not a page). In that window you can work with different pages and different configuration settings.

File → Close Window (F C) closes the currently active lepton-schematic window (the window where you use this shortcut) with all its pages. The Save File dialog box will be open for every unsaved page. See Saving schematic pages.
