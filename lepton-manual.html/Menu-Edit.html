5.3.3 Menu Edit

Undo

Edit → Undo (Ctrl-Z, U, Shift-U, or E Shift-U) undoes the last editing action. See Editing schematics.

Redo

Edit → Redo (Shift-Ctrl-Z, Ctrl-Y, Shift-R, or E Shift-R) repeats recently undone actions. See Editing schematics.

Cut

Edit → Cut (Ctrl-X) cuts currently selected objects to clipboard. See Editing schematics.

Copy

Edit → Copy (Ctrl-C) copies currently selected objects to clipboard. See Editing schematics.

Paste

Edit → Paste (Ctrl-V) allows to paste the current clipboard contents to the drawing area. See Editing schematics.

Delete

Edit → Delete (Delete, D, or E D) deletes currently selected objects. See Editing schematics.

Select Mode

Edit → Select Mode (S or E S) switches on the selection mode. See Selecting objects.

Select All

Edit → Select All (Ctrl-A) selects all non-locked objects on a page. See Selecting objects.

Deselect

Edit → Deselect (Shift-Ctrl-A) deselects all selected objects, if any. See Selecting objects.

Copy Mode

Edit → Copy Mode (C or E C) allows one-time copy of selected objects. See Editing schematics.

Multiple Copy Mode

Edit → Multiple Copy Mode (E Y) allows to make several copies of the same objects. See Editing schematics.

Move Mode

Edit → Move Mode (M or E M) allows you to move selected objects without depressing the left mouse button. See Editing schematics.

Rotate 90 Mode

Edit → Rotate 90 Mode (E R) rotates selected objects by 90 degrees counter-clockwise each time. See Editing schematics.

Mirror Mode

Edit → Mirror mode (E I) is used to mirror objects around the X axis. See Editing schematics.

Object Properties...

Edit → Object Properties... (E O) opens a dialog where you can see and change the properties of primitive objects. See Editing schematics.

Edit...

Edit → Edit... (E E) opens one of the editing dialogs, Edit Attributes, Edit Attribute, or Edit Text, depending on what objects are selected. See Editing schematics.

Edit Text...

Edit → Edit Text... (E X) opens the Edit Text dialog box for editing plain text objects. See Editing schematics.

Slot...

Edit → Slot... (E Shift-S) is used to change slot number of a slotted component. See Editing schematics.

Lock

Edit → Lock (E L) locks selected objects, that is, makes them not selectable by just clicking on them. See Editing schematics.

Unlock

Edit → Unlock (E Shift-L) unlocks selected locked objects. See Editing schematics.

Embed Component/Picture

Edit → Embed Component/Picture (E B) embeds a component or a picture into schematic. See Editing schematics.

Unembed Component/Picture

Edit → Unembed Component/Picture (E Shift-B) unembeds an embedded component or a picture from schematic. See Editing schematics.

Update Component

Edit → Update Component (E U) updates selected components if they changed before. See Editing schematics.

Symbol Translate...

Edit → Symbol Translate... (E T) translates a symbol being edited to origin. See Editing schematics.
