1.1 The list of programs in the suite

Lepton EDA suite includes tools as follows:

  • lepton-schematic—the main GUI tool for schematic capture
  • lepton-attrib—spreadsheet schematic attribute editor
  • lepton-netlist—a tool for making netlists from command line
  • lepton-cli—a CLI tool providing support of configuration, export, and shell
  • lepton-config—configure your tools
  • lepton-export—export schematics to images and multipage documents
  • lepton-shell—interactive processing of schematic data in Scheme REPL
  • lepton-archive—project archivation utility
  • lepton-tragesym—symbol generator
  • lepton-embed—embed symbols and pictures into schematics
  • lepton-pcb_backannotate—backannotate changes from PCB to schematic
  • lepton-refdes_renum—renumber reference designators
  • lepton-renum—an advanced refdes renumber utility
  • lepton-sch2pcb—generate PCB layout from a set of schematics
  • lepton-schdiff—view graphical diff of schematics
  • lepton-symcheck—check symbol files for errors
  • lepton-symfix—fix attributes in symbol files
  • lepton-upcfg—upgrade gEDA config files to be usable in Lepton
  • lepton-update—update obsolete schematics to the current version

The following contributed tools are not covered in this manual. Mostly, those are for working with proprietary, often obsolete, programs, so they are of limited use for most users. Please see corresponding man pages for more information.

  • convert_sym—convert a Viewlogic symbol/schematic to gEDA/gaf File Format Document.
  • gmk_sym—create rectangular symbols in gEDA/gaf File Format Document from a text file.
  • olib—OrCAD(tm) library part to gEDA .sym converter.
  • sarlacc_schem—convert OrCAD SDT IV files to gEDA/gaf File Format Document.
  • smash_megafile—break a Viewlogic megafile into a million little pieces.
  • gnet_hier_verilog.sh—generate a non-flattened hierarchical Verilog netlist.
  • pads_backannotate—process PADS PowerPCB .eco files to backannotate changes to schematics in gEDA/gaf File Format Document.
  • sw2asc—converts a SWITCAP2 output file into ASCII data files that other tools can read. See gEDA/gaf Switcap Symbols and Netlister for more information.
  • sch2eaglepos.sh—read a schematic in gEDA/gaf File Format Document and attempt to extract the relative positions of the components in order to generate corresponding MOVE instructions for Eagle.
  • sarlacc_sym—convert OrCAD text libraries to components in gEDA/gaf File Format Document.
